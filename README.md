# 🌟 Employee Management System | Leave Tracker

🔗 **Live Demo:** https://employee-management-sysytem-leave.onrender.com/

---

## 📌 Project Overview

It is a full-stack Employee Leave Management System built using Django.  
It automates the leave application and approval process, providing a seamless experience for both employees and administrators.

---

## 🚀 Features

### 👨‍💼 Employee
- Register and login securely  
- Apply for leave  
- View leave history and status (Pending / Approved / Rejected)  
- Clean and responsive dashboard  

### 🛠️ Admin
- Secure admin login  
- View all employees  
- Approve / Reject leave requests  
- Delete leave requests  
- Real-time leave status updates  

### 📧 Email Notifications
- Automatic email sent on:
  - Registration  
  - Leave approval/rejection  
- Integrated using Gmail SMTP  

---

## 🧑‍💻 Tech Stack

- **Backend:** Django, Python  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite3  
- **Email Service:** Gmail SMTP
